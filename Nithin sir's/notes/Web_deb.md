##  HTML Structure:
	
* Arranging the elements of HTML in a hierarchical way to present on a webpage is called "__DOM__" .
```html
<html> 
    <!-- Document scope-->  
    <head>

    </head> 
    <body>

    </body> 
</html>
```

*   The content present in HEAD section is mainly use for SEO and responsive design.

*   The elements which written inside the "head" section are :
	* title
	* link	
	* metadata
	* style 
	* script


*  It is a indication to a browser engine that in the body section we use element of HTML5.
			<html lang="en-IN"* 

*  it is indication to a browser engine that the output should be feasible to the end users in particular format.
